```
- Drizzle: https://orm.drizzle.team/llms.txt
- shadcn: https://ui.shadcn.com/llms.txt

Build a...
```

### With AI Tools
- Claude: Paste URL in prompt
- ChatGPT: Use in custom GPT instructions
- MCP: Use web fetch tool

### For AI Builders
```ts
// Fetch and parse
const response = await fetch('https://nextjs.org/llms.txt')
const docs = await response.text()
// Add to RAG system
```

## Why llms.txt?

### Benefits
- LLMs can reference latest docs
- Always up-to-date info
- Reduces hallucinations
- Standardized format
- Easy to maintain

### Use Cases
- AI coding assistants
- Documentation chatbots
- RAG systems
- Custom GPTs
- AI agents

## Notes
- Not all projects have llms.txt yet
- Format is evolving
- Some use `/docs/llms.txt` path
- Check project docs if URL doesn't work
