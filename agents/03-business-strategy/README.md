# 03 Business Strategy

Agents for validating ideas, planning go-to-market, and optimizing monetization.

## Agents

| Agent | Model | Description |
|-------|-------|-------------|
| [idea-validator](./idea-validator.md) | Opus | Market validation using David Andre framework |
| [product-strategist](./product-strategist.md) | Opus | Product vision, roadmap, feature prioritization |
| [gtm-strategist](./gtm-strategist.md) | Opus | Go-to-market planning, launch strategy |
| [growth-hacker](./growth-hacker.md) | Sonnet | User acquisition, SEO, conversion optimization |
| [marketing-expert](./marketing-expert.md) | Sonnet | Marketing tactics, messaging, campaigns |
| [monetization-advisor](./monetization-advisor.md) | Opus | Pricing strategy, Stripe integration, revenue |
| [project-planner](./project-planner.md) | Sonnet | Task breakdown, milestones, roadmapping |

## When to Use

- **New idea**: Use `idea-validator` first to validate market fit
- **Product planning**: Use `product-strategist` for roadmap and features
- **Launch prep**: Use `gtm-strategist` for go-to-market strategy
- **User growth**: Use `growth-hacker` for acquisition tactics
- **Marketing**: Use `marketing-expert` for campaigns and messaging
- **Pricing**: Use `monetization-advisor` for pricing models
- **Task planning**: Use `project-planner` for breaking down work

## Recommended Flow

```
idea-validator → product-strategist → project-planner → [build] → gtm-strategist → growth-hacker
```
