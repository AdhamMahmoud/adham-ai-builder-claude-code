# 01 Core Development Agents

Specialized agents for building web applications with Next.js, React, and modern tooling.

## Agents

| Agent | Model | Description |
|-------|-------|-------------|
| [nextjs-developer](./nextjs-developer.md) | Sonnet | Next.js 16 App Router, RSC, Server Actions, Turbopack |
| [react-developer](./react-developer.md) | Sonnet | React 19 patterns, hooks, state management |
| [fullstack-developer](./fullstack-developer.md) | Opus | End-to-end across database, API, frontend layers |
| [api-architect](./api-architect.md) | Opus | REST/GraphQL/tRPC API design and implementation |
| [database-architect](./database-architect.md) | Opus | Drizzle ORM, PostgreSQL, schema design |
| [ui-designer](./ui-designer.md) | Sonnet | shadcn/ui, Tailwind CSS, accessible components |

## When to Use

- **New feature development**: Use `fullstack-developer` for end-to-end implementation
- **Frontend only**: Use `react-developer` or `nextjs-developer` depending on framework needs
- **API design**: Use `api-architect` for REST/GraphQL patterns and contracts
- **Database schema**: Use `database-architect` for data modeling and migrations
- **UI components**: Use `ui-designer` for visual components and design systems

## Related Skills

- `nextjs-patterns` - App Router, Server Components, caching strategies
- `react-patterns` - React 19 hooks, Suspense, Error Boundaries
- `database-patterns` - Drizzle ORM queries, relations, migrations
- `ui-library` - shadcn/ui components, Radix primitives, Tailwind utilities
- `auth-patterns` - NextAuth v5, OAuth, session management
